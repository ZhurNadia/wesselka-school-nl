## 🎨 Дизайн

### Кольорова схема
```
--blu:  #1B4FBE   (основний синій)
--blu2: #0f3490   (темний синій)
--yel:  #F5C842   (жовтий — акцент)
--hon:  #E8A020   (медовий)
--pch:  #FDE8CC   (персиковий — фон карток)
--crm:  #FFFBF4   (кремовий — фон сторінки)
```

### Шрифти
- **Playfair Display** — заголовки (serif)
- **Nunito** — основний текст (sans-serif)

### Hero-візуал (oval-карусель)
- Овальна рамка з CSS `border-radius` морфінг-анімацією (`morphBorder`, 8s loop)
- 5 плаваючих бульбашок з emoji (🌈🇺🇦✨📚🎵), анімація `floatBubble`
- Карусель всередині: 3 фото, автопрокрутка 3.8 сек, свайп на мобільному
- На екранах < 1000px oval прихований, контент займає повну ширину

## ⚙️ Функціональність

### Локалізація (I18N)
- 3 мови: **UA / NL / EN**
- Перемикач у навігації (десктоп) та мобільному меню
- Перекладається **весь** текст сайту: навігація, hero, про нас, описи карток, розклад, заходи, форма, FAQ, контакти, футер
- Реалізовано через об'єкт `i18n` + `data-i18n` атрибути + функція `setLang()`

### Календар субот 2026
Кольорове кодування:
- 🟠 Канікули (`#fde8cc`)
- 🔵 Свята/вихідні (`#e8f0fe`)
- 🟢 Особливі події (`#e8fce8`) — напр. 06/06 🎉 15 років школі
- ⬜ Навчання (білий)

### Карта (Google Maps)
- Без API ключа — стандартний `iframe` embed через `google.com/maps?q=...&output=embed`
- **Дві локації перемикаються по кліку на картку зліва** (не окремі вкладки)
- Активна картка підсвічується синім
- Підказка "👆 Натисніть на локацію" (перекладена на 3 мови)

### Форма реєстрації
- Вбудована через `<iframe>` з Google Forms
- URL: `https://docs.google.com/forms/d/e/1FAIpQLSeWM9Y8k8p1vt7oekHgyD0Qhr1wPzguv5mA2KEG5E1joQfTaQ/viewform?embedded=true`
- Підказка про Google-акаунт + запасне посилання `forms.gle/y5wt8CvtMnz4p5Rq5`

### Розкривані картки "Про нас"
- **Мова, культура та творчість** — традиційні свята, майстер-класи, ярмарок, 2 фото
- **Клуб «Читанка»** — опис, фото, цитата-кредо, засновниця Наталія Самборук

### Модальне вікно реєстрації на заходи
- Відкривається по кнопці "Записатись" у картках подій
- Поля: ім'я, email, кількість людей

## 📝 Нотатки для Claude Code

Якщо продовжуєш роботу в Claude Code — ось ключові деталі:

1. Весь сайт — **один файл** `veselka-school.html`
2. Зображення вбудовані як `data:image/webp;base64,...` — для заміни фото треба перекомпресувати через Python + PIL
3. Переклади — об'єкт `i18n` наприкінці `<script>` тегу, ключі `uk/nl/en`
4. Карта перемикається функцією `switchMap('fr')` / `switchMap('co')`
5. Oval-карусель — функція `ovalGo(n)`, автозапуск `ovalAuto()`
6. Scroll reveal — клас `.fu` + `.vis` через IntersectionObserver
